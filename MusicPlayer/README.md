
# Music Player Web Application

## Overview

This project is a web-based music player created with HTML, CSS, and JavaScript. It provides a clean, user-friendly, and responsive interface for playing audio files, managing playlists, and controlling playback. The application is designed to offer a seamless and enjoyable user experience.

## Features

- **Clean User Interface**: A modern and intuitive design that is easy to navigate.
- **Audio Playback**: Core functionality to play, pause, and seek through audio tracks using the HTML5 `<audio>` element.
- **Playlist Management**:
  - **Upload Music**: Users can upload their own audio files to create a personal playlist.
  - **Dynamic Playlist**: The playlist is displayed in the sidebar, showing the song title, artist (if available), and duration.
  - **Playlist Controls**: Options to shuffle the playlist, sort it alphabetically, or clear all tracks.
- **Playback Controls**:
  - **Standard Controls**: Play, pause, next song, and previous song.
  - **Repeat**: Cycle through repeat modes (off, repeat all, repeat one).
  - **Progress Bar**: A visual progress bar that shows the current playback time and total duration. Users can click on the bar to seek to a specific part of the song.
- **Volume Control**: A volume slider to adjust the audio level and a mute button to toggle sound on and off.
- **Song Information**: Displays the title, artist, and album of the currently playing song.
- **Responsive Design**: The layout adapts smoothly to different screen sizes, ensuring a great experience on desktops, tablets, and mobile devices.
- **Visualizer**: A simple animated audio visualizer that appears when a song is playing.
- **Keyboard Shortcuts**: Control the player using your keyboard (Spacebar for play/pause, arrow keys for navigation and volume).
- **Notifications**: On-screen notifications for actions like adding a song, shuffling, or clearing the playlist.

## How to Use the Music Player

1.  **Open the Application**: Launch the `index.html` file in a web browser.
2.  **Upload Music**: Click the **Upload** button (cloud icon) in the top-right corner to select one or more audio files from your computer.
3.  **Manage Your Playlist**:
    - Once uploaded, your songs will appear in the playlist on the left.
    - You can **rename** the playlist by editing the text field at the top of the sidebar.
    - Use the **Shuffle** and **Sort** buttons to organize your playlist.
    - Click the **trash icon** to clear the entire playlist.
4.  **Play Music**:
    - Click the **play icon** next to any song in the playlist to start playing it.
    - Use the main controls at the bottom of the player to play, pause, or skip tracks.
5.  **Control Playback**:
    - Drag the **progress bar** to seek to a different part of the song.
    - Use the **volume slider** to adjust the volume.

## Technologies Used

-   **HTML5**: For the structure and layout of the music player.
-   **CSS3**: For styling, animations, and responsive design.
-   **JavaScript (ES6)**: For all the player's functionality, including audio control, playlist management, and DOM manipulation.
-   **Font Awesome**: For icons used throughout the user interface.
